# 📐 高校数学可視化：ベクトルの内積と動的なす角

TypeScript と Canvas API (Vite) で構築された、インタラクティブな幾何学・ベクトルシミュレーターです。
高校数学で学習する「ベクトルの成分」「内積の計算」「なす角（$\theta$）の変化」を、マウス操作によって直感的にリアルタイムで理解・体験することができます。

## 🚀 特徴

- **直感的なドラッグ操作**: ベクトル $\vec{a}$ (赤) と $\vec{b}$ (青) の先端をマウスで直接掴んで動かせます。
- **リアルタイム数学計算**: 座標成分、内積の値、なす角（度数法）が即座に計算され、画面に反映されます。
- **視覚的な条件判定**: 内積の正負による「鋭角」「垂直」「鈍角」の状態変化を、色付きのバッジで分かりやすく通知します。

## 🛠️ 技術スタック

- **言語**: TypeScript
- **描画**: HTML5 Canvas API
- **ビルドツール**: Vite
- **デザイン**: プレーンCSS (モダンUIスタイリング)

起動後、ターミナルに表示されるローカルURL（例: http://localhost:5173/）にブラウザでアクセスしてください。💡 このシミュレーターで学べる数学的発見この教材は、数式だけでは実感が湧きにくい以下のベクトル・幾何学の性質を視覚的に証明・体感できるように設計されています。内積が「正」のとき（$\vec{a} \cdot \vec{b} > 0$）2つのベクトルは同じ方向の成分を持ち、なす角は 鋭角（$0^\circ \le \theta < 90^\circ$） になります。内積が「0」のとき（$\vec{a} \cdot \vec{b} = 0$）2つのベクトルは完全に 直交（$\theta = 90^\circ$） します。高校数学の証明問題で最頻出の「垂直 $\Leftrightarrow$ 内積0」を体験できます。内積が「負」のとき（$\vec{a} \cdot \vec{b} < 0$）2つのベクトルは互いに背を向ける方向の成分を持ち、なす角は 鈍角（$90^\circ < \theta \le 180^\circ$） になります。📝 ライセンスMIT License
